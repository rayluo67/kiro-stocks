# 技术栈

## 后端技术
- **框架**: Flask (Python)
- **包管理**: uv (用于创建虚拟环境和管理Python包)

## 前端技术
- **框架**: React

## 开发环境
- **终端**: Fish Shell
- **Python环境管理**: uv

## 常用命令

### 环境设置
```bash
# 创建虚拟环境
uv venv

# 激活虚拟环境
source .venv/bin/activate

# 安装依赖
uv pip install -r requirements.txt
```

### 开发运行
```bash
# 启动后端服务
uv run app.py

# 启动前端开发服务器
npm start
# 或
yarn start
```

## 编码规范

### 语言使用
- **程序代码**: 使用英文编写变量名、函数名、类名等
- **注释和文档**: 使用中文编写
- **API文档**: 使用中文描述

### 代码风格
- Python代码遵循PEP 8规范
- React代码使用ES6+语法
- 使用有意义的英文命名
